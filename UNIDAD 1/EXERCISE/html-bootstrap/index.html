Bienvenidos

Este proyecto demuestra el uso de etiquetas HTML, hojas de estilo CSS y componentes de Bootstrap en una página web simple y moderna.

CSS

Estilos personalizados para colores, tipografías, espaciado y efectos visuales.

Bootstrap

Componentes responsivos como navbar, grid, cards y formularios.
