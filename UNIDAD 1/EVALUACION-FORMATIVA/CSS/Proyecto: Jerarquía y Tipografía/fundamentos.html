Cómo piensa el navegador cuando aplicas CSS

Antes de crear diseños complejos, conviene dominar los conceptos que hacen que todo encaje: selección, medida y espacio.

Selectores Unidades Box Model Legibilidad
01

Selectores básicos

Los selectores son la forma de decirle al navegador qué elemento quieres estilizar.

Etiqueta: afecta a todos los elementos del mismo tipo.

Clase: reutilizable para varios elementos.

ID: identificador único para un elemento específico.

02

Unidades de medida

px ofrece control fijo, pero no escala tan bien para accesibilidad. rem se adapta al tamaño base del usuario y permite interfaces más flexibles.

Regla práctica: usa rem para tipografía y espaciados generales, y reserva px para detalles muy concretos.

03

Box Model

Todo elemento es una caja compuesta por: content, padding, border y margin.

Si algo "no cabe" o se ve desalineado, normalmente la respuesta está en cómo se suman estas cuatro capas.

04

Jerarquía visual y lectura cómoda

No basta con colores bonitos: una buena jerarquía guía la mirada del usuario con títulos claros, contraste y espacio en blanco.

Para textos de lectura continua, un line-height entre 1.5 y 1.7 suele funcionar muy bien.
