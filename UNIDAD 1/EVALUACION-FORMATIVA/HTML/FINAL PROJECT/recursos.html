Guía de Recursos de GitHub

Tal como se indica en el módulo, aquí tienes los pasos para subir tu proyecto local a un repositorio.


Pasos para subir tu código

1
Crea un nuevo repositorio: Ve a tu cuenta de GitHub y crea un repositorio público llamado html-final-project.
2
Inicializa tu carpeta: Abre la terminal en tu carpeta local y escribe git init para empezar el seguimiento.
3
Prepara tus archivos: Usa git add . para incluir todos tus archivos locales.
4
Sube el proyecto (Push): Vincula tu repositorio remoto y usa git push origin main para completar la entrega.
